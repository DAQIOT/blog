### Core challenge: How to achieve efficient collection of EDMI meters?
**How to achieve data monitoring of EDMI Protocol electric meter (Mk6E) without disassembling or changing the wiring, in the face of special collection requirements?**
### Difficulties in project implementation
Electricity meter restriction: On site electricity meters are prohibited from dismantling and connecting wires
Interface limitation: Only optical communication port (infrared communication) is retained
Protocol Special: Unconventional State Grid Protocol, difficult to parse
• Clear requirements: Low cost, easy wiring, wireless transmission
