## 1 **Product Overview**
![图片](https://dev-to-uploads.s3.us-east-2.amazonaws.com/uploads/articles/ib055cqtxhapp3hlxpu5.jpg)
The purpose of this software is to monitor the running of exe programs on computers or servers. When the exe program exits due to abnormalities or other reasons, it will automatically start the exe program. It has a self startup function and can automatically run the monitoring program itself and start the monitored exe program after booting up
## 2 **software installation**
This software is a pure green software that does not require installation. Simply unzip it to any location on the system to run.
