# 2 **project Demand research analysis**
Requirement analysis: Within the Qingming Shanghe Garden Scenic Area,[Da Song School Field](https://zhida.zhihu.com/search?content_id=260528516&content_type=Article&match_order=1&q=%E5%A4%A7%E5%AE%8B%E6%A0%A1%E5%9C%BA&zhida_source=entity)The surrounding observation deck is built in the style of Song Dynasty architecture. To ensure safety in the scenic area, it is necessary to monitor the load-bearing columns and main beams of the observation deck to prevent safety hazards caused by the displacement of the main beams. Eight points on site need to be monitored for displacement.

# 3 **Project Proposal**
adopt[4G laser rangefinder](https://zhida.zhihu.com/search?content_id=260528516&content_type=Article&match_order=1&q=4G%E6%BF%80%E5%85%89%E6%B5%8B%E8%B7%9D%E4%BB%AA&zhida_source=entity)The Hardware collection scheme indirectly monitors the deviation of the main beam by monitoring the distance between the main beam and the fixed position on the ground, which is in line with the actual cost and construction aspects.
Considering that wiring cannot be done on site, the laser rangefinder is powered by batteries. Laser rangefinders are installed at 8 points on the main beam to measure and monitor whether the main beam position has moved downwards. The collected data is transmitted to the platform via 4G to facilitate customer viewing of the data.
